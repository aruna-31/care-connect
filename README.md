# CareConnect Live

## Project Overview
CareConnect Live is a real-time doctor-patient consultation platform featuring role-based dashboards, appointment workflows, and secure chat consultations.

## Quick Start
### Frontend
1. `cd frontend`
2. `npm install` (if not already done)
3. `npm run dev`
4. Visit `http://localhost:5173`

### Backend
1. `cd backend`
2. `npm install`
3. Ensure PostgreSQL is running and `.env` is configured.
4. `npm run dev` (starts on port 8000)
